# Video-based-Action-Recognition-using-GluonCV
Requirements:
Python 3.6
pip >= 9
mxnet
PyTorch - torchvision
GluonCV

Installation steps:
# for mxnet
pip install --upgrade mxnet
# for pytorch
pip install torch==1.6.0+cpu torchvision==0.7.0+cpu -f https://download.pytorch.org/whl/torch_stable.html
# for GluonCV
pip install --upgrade gluoncv
